
介绍
----
使用 `node.js` 和 `socket.io` 写的的一个简单聊天室插件，本人小白练习所以功能比较少，可实现多人在线聊天、显示用户昵称、聊天内容以及发表时间，暂时只支持输入文字内容，更多功能还有待更多程序猿开发呐！

效果图：




安装
----
```sh
    $ git clone https://github.com/Fanxinheng/chatroom.git 
```
```
    $ cd chatroom
```
```sh
    $ npm install
```


使用
----
- 1
```sh
    $ node chatroom.js
```
- 2

浏览器地址栏输入 `localhost:3000`
